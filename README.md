# Decision Tree

This is a program that trains and tests a decisionn tree to classify spam or not spam from the spambase dataset. It trains the decision tree using the ID3 algorithm without any pruning.

## Usage

`python3 tree.py`

## Dependencies

- `python 3.8+`

### Python Dependencies

- `numpy`
- `matplotlib`
- `scipy`
- `sklearn`

## Results

- Accuracy: `90.41%`
- Precision: `87.88%`
- Recall: `88.31%`
- F-measure: `88.10%`
